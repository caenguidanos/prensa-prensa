![Diagram](./static/workspace-diagram.svg)

## Requirements

-  node: `16.13.0`
-  golang: `+1.17`
-  nvm
-  pnpm

## Installation

```bash
pnpm install
```

## Run

> `start` doesn't works if you not provides custom environment **variables** in script

```
pnpm all:dev
pnpm all:lint
pnpm all:test
pnpm all:build
pnpm all:start
pnpm all:e2e
```
